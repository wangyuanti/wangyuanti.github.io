v-if

v-if=”数据”
条件->true||false
如果条件成立则显示

v-else-if

v-else-if=”数据”
条件->true||false

v-else

必须紧跟这v-if或者v-else-if

v-for

不但能循环数组也能循环对象
v-for=”val in arr”
v-for=”(val,key) in arr”

v-text=”数据”

等同于innerText

v-html=”数据”

等同于innerHTML

v-on

v-on:click=”函数名”
简写@click=”函数名”
@keydown
@mouseover
@mouseout
….
事件函数必须报在methods下，它为一个对象

<div id="root">
    <!-- <button v-on:click="changeNum">按钮</button> -->
    <button @click="changeNum">按钮</button>
    <!-- <button @mouseover="changeNum">按钮</button> -->
    <ul>
        <li v-for="(val,key) in arr">{{val}}</li>
    </ul>
</div>
<script src="vue.js"></script>
<script>
let v = new Vue({
        el:'#root',
        data:{
           arr:[],
           num:0
        },
        methods:{
            changeNum(){
                let {num} = this;
                num ++;
                this.num = num;
                this.arr.push(num);
            }
        }
    });
</script>

.prevent - 调用 event.preventDefault()。

.once 只能点一次

v-bind

一般用在操作行间数据
:class=”bg”

{key:变量}

v-model

专门用来操作表单的
!!!如果使用数组循环，那么要启用v-model,要写
数组[key] 不能写val 

arr = [true,false]
<input 
type="checkbox" 
v-for="(val,key) in arr" 
v-model="arr[key]"    
/>

!!!不要用val，要使用arr[key]

v-model.lazy=”val”

失焦的时候会触发

v-model.number type=”number”

只认数字


去掉前后空格

<input type="text" v-model="val"/>
{{val}}
<input type="radio" value="0" v-model="val2"/>
<input type="radio" value="1" v-model="val2"/>
{{arr3[val2]}}

v-show

在if和show之间更建议使用show，性能会好些

computed

当依赖的数据发生变化的时候，不想改变原来的数据
又想形成一个新的数据时使用
方法和computed区别
方法调用一次执行一次
计算属性，调多少次都执行一次

watch

一上来是不执行的，只有监听的数据发生变化才执行
