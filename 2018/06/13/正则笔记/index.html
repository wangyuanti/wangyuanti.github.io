正则表达式的笔记

发表于 2018-06-13 | 分类于 笔记

正则： 专门操作检索字符串的规则工具

有两种写的方式：

/字符串/ 简写 里面只能为字符串
new Regexp(正则||字符串) 标准写法

\转义符
\ + 一些字母，有可能会为一个特殊的符号

正则方法

.test(string) 返回的是布尔值
如果string中有正则匹配项就返回true，否则false

###字符串方法
string.search(‘’||reg) 返回下标或者-1
找到指定字符的位置 没有找到返回-1
与indexOf类似

string.match(//) 返回[]或null
找到匹配项，返回的是一个数组 没有找到返回null

string.replace(要替换的|//,替换成什么|callback) 新的字符串
callback:默认 必须要有返回值，不然都为undefined
每次匹配到数据之后会调用一次这个函数
callback的参数为$0、$1、$2、$3等等
一般来说：
第一个参数: 每次匹配到的数据
第二个参数: 每次找到数据的下标
第三个参数: 整个字符串
第四个参数（包括之后）: undefined
例子：

let str = '中国共产党总书记习近平说:"法轮功是邪教"';
let s = str.replace(/共产党|习近平|法轮|邪/g,function($0,$1,$2,$3,$4){
    let temp = '';
    for(let i=0;i<$0.length;i++){
         temp += '*'; 
    }
    return temp;
});
console.log(s);

修饰符

i -> 忽略大小写
g -> 修饰符 找全局

其他一些东西

量词:
{a,b}
a最小，b最大
‘+’ {1,}最小一个，最多无限
‘?’ {0,1}最小没有，最多一个
‘*’ {0，}最小没有，最多无限
例子：

let str = 'a232132131ca5432ca89caca8c';
console.log(str.match(/a\d*c/g));

或者：|

子项：()
1.在正则中加括号,如果使用repalce，那么从第一参数之后就是子项
有几个子项，就会对应回调函数的参数 当子项用完，又是index，all，undefined
2.提权 (1+2)*3
例子：

let str = '2018-05/////////////17';
 let s = str.replace(/(\d+)\D+(\d+)\D+(\d+)/,function($0,$1,$2,$3){
     return $1 +"年" + $2 + '月' + $3 + '日';
 });
 console.log(s);

关于子项的经典面试题：
使用参数$1来找到对象里对应的数据

let obj = {
    name:'肾肾',
    age:16,
    j:'前端开发',
    "2":'呵呵'
}
let str = '我的名字叫${name},年龄有${age}岁,职业:${j},${2}';
let s = str.replace(/\${(\w+)}/gi,function($0,$1){
    return obj[$1];
});
console.log(s);

重复子项(子项重定向)
\number
console.log(str.match(/a(3)\1(d)\2(g)\3+c/g));
a开头c结尾
中间为
(3)\1 第一个子项是3 \1 重复第一个子项3 == 33
(d)\2 第二个子项是d \2 重复第二个子项d == dd
(g)\3 第三个子项是g \3+ 重复第三个子项g == ggg

开头：^

结尾：$

不知道怎么形容：[]
在中括号中取其一
可以用来规定范围
小写字母：a-z
数字:0-9
大写字母：A-Z
大小写字母：A-Za-z
汉字：\u4e00-\u9fa5
例子：

let str2 = '200';
console.log(/^1[89]$|^[2-9][0-9]$|^10[0-9]$|^110$/.test(str2));

计算字符的面试题：
请计算下列字符串的字节长度
假设：数字、字母为1个字节，中文2个字节

let str = '你好!mynameis_马虎。'; //17个;
let num = 0;
for(let i=0;i<str.length;i++){
    if(/[\u4e00-\u9fa5]/.test(str[i])){
        num += 2;
    }else{
        num ++;
    }
}
console.log(num);

排除：

let str = 'a1ca2ca4cabcagca c';;
console.log(str.match(/a[^4a-z\s]c/g));

\d 一个数字
\D 一个非数字
\w 一个字母数字下划线
\W 一个非字母数字下划线
\s 一个空格
\S 一个非空格
\b 边界符
\B 非边界符
英文左右两边如果是开头或结尾或空格都算边界符
中文没有边界符的
例子：

let str = '我dss  sa d我 a';
console.log( str.replace(/\b/g,'|'));
//我|dss|  |sa| |d|我 |a|
